### 自动化及其他

## 使用

-   启动开发

```
yarn dev
```

-   启动打包

```
yarn build
```

### 使用 hygen

-   创建一个组件

```
yarn hygen generator component Card
```

-   创建一个页面

```
yarn hygen generator pages user
```
